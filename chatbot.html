Chatbot

Object Description:

The Chatbot object handles user input and generates responses by interacting with other components of the system. It utilizes the Microphone, Memory, Responses, and Trigger objects to provide a seamless user experience.

Main Functionalities:

  • Listen to the user's input through the Microphone object
  • Detect the trigger word "Giz" using the Trigger object
  • Generate a response using the Responses object
  • Store the conversation history in the Memory object

Methods:

  • __init__(self, microphone, memory, responses, trigger): Initialize the Chatbot object with instances of Microphone, Memory, Responses, and Trigger objects.
  • listen(self): Listen to the user's input and return the transcribed text.
  • detect_trigger(self, text): Detect the trigger word in the given text and return a boolean value.
  • generate_response(self, text): Generate an appropriate response based on the given text.
  • store_conversation(self, input_text, response_text): Store the conversation history in the Memory object.
  • run(self): Run the chatbot loop, listening for user input, detecting the trigger, generating responses, and storing conversation history.
